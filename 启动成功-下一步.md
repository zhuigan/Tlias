## ⚠️ 注意事项

### 关于阿里云OSS

如果图片上传成功但无法显示，需要配置阿里云OSS：

1. 登录阿里云控制台：https://oss.console.aliyun.com/
2. 找到你的 Bucket
3. 设置权限为"公共读"
4. 配置 CORS 跨域规则

详细步骤请查看：`阿里云OSS配置指南.md`

### 本地存储备用方案

如果阿里云OSS配置有问题，系统会自动使用本地存储：
- 图片保存在项目的 `uploads/` 目录
- URL 格式：`http://localhost:8080/uploads/xxx.jpg`
- 可以正常显示和使用

## 🔍 故障排查

### 如果登录后没有跳转

1. 检查浏览器控制台是否有错误
2. 确认 token 是否正确生成
3. 查看后端日志

### 如果头像无法显示

1. 查看上传日志（访问 test-upload.html）
2. 检查图片 URL 是否正确
3. 如果是阿里云OSS，检查 Bucket 权限
4. 如果是本地存储，检查 uploads 目录是否存在

### 如果权限不正确

验证数据库：
```sql
SELECT username, name, role FROM emp WHERE username IN ('admin', 'zhangwuji');
```

应该显示：
- admin: role = 1
- zhangwuji: role = 2
