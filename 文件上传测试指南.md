# 文件上传功能测试指南

## 📋 功能说明

这个接口用于上传图片到阿里云 OSS 存储，返回图片的访问 URL。

## 🔧 后端接口信息

### 接口地址
```
POST http://localhost:8080/upload
```

### 请求参数
- **参数名**：`image`（必须是这个名字）
- **参数类型**：`MultipartFile`（文件类型）
- **Content-Type**：`multipart/form-data`

### 响应格式
```json
{
  "code": 1,
  "msg": "success",
  "data": "https://oss-cn-hangzhou.aliyuncs.com/bucket/uuid-filename.jpg"
}
```

## 🚀 测试方法

### 方法1：使用 Postman（推荐）

#### 步骤1：打开 Postman
下载地址：https://www.postman.com/downloads/

#### 步骤2：创建新请求
1. 点击 "New" → "HTTP Request"
2. 或者点击 "+" 创建新标签页

#### 步骤3：配置请求
1. **请求方法**：选择 `POST`
2. **请求 URL**：输入 `http://localhost:8080/upload`

#### 步骤4：设置请求体
1. 点击 **Body** 标签
2. 选择 **form-data**（这是关键！）
3. 在 Key 列输入：`image`
4. 在 Key 右侧的下拉菜单中选择：**File**（不是 Text）
5. 在 Value 列点击 "Select Files" 按钮
6. 选择一张图片文件（jpg、png、gif 等）

#### 步骤5：添加认证（如果需要）
如果接口需要登录认证：
1. 点击 **Headers** 标签
2. 添加：
   - Key: `token`
   - Value: `你的JWT Token`

#### 步骤6：发送请求
1. 点击右上角的 **Send** 按钮
2. 查看下方的响应结果

#### 预期响应
```json
{
  "code": 1,
  "msg": "success",
  "data": "https://oss-cn-hangzhou.aliyuncs.com/hmleadnews1111111111111111111111111/uuid-image.jpg"
}
```

## ⚠️ 注意事项

### 1. 阿里云 OSS 配置

当前代码中的阿里云 OSS 配置是硬编码的：

```java
private String endpoint = "https://oss-cn-hangzhou.aliyuncs.com";
private String accessKeyId = "aliyun.oss.accessKeyId";
private String accessKeySecret = "aliyun.oss.accessKeySecret";
private String bucketName = "hmleadnews1111111111111111111111111";
```

**问题**：
- 这些是示例配置，可能已经失效
- AccessKey 暴露在代码中不安全

**建议**：
1. 使用自己的阿里云 OSS 账号
2. 将配置移到 `application.properties`
3. 不要将 AccessKey 提交到 Git

### 2. 修改为配置文件方式

#### 修改 application.properties
```properties
# 阿里云 OSS 配置
aliyun.oss.endpoint=https://oss-cn-hangzhou.aliyuncs.com
aliyun.oss.accessKeyId=你的AccessKeyId
aliyun.oss.accessKeySecret=你的AccessKeySecret
aliyun.oss.bucketName=你的BucketName
```

#### 修改 AliOSSUtils.java
```java
@Component
public class AliOSSUtils {

    @Value("${aliyun.oss.endpoint}")
    private String endpoint;
    
    @Value("${aliyun.oss.accessKeyId}")
    private String accessKeyId;
    
    @Value("${aliyun.oss.accessKeySecret}")
    private String accessKeySecret;
    
    @Value("${aliyun.oss.bucketName}")
    private String bucketName;

    // ... 其他代码不变
}
```

### 3. 如果没有阿里云 OSS

如果你没有阿里云 OSS 账号，可以：

#### 选项1：使用本地文件存储
修改 `FileController.java`，将文件保存到本地：

```java
@PostMapping("/upload")
public Result upload(MultipartFile image) throws IOException {
    // 保存到本地目录
    String uploadDir = "uploads/";
    String fileName = UUID.randomUUID() + "_" + image.getOriginalFilename();
    File dest = new File(uploadDir + fileName);
    dest.getParentFile().mkdirs();
    image.transferTo(dest);
    
    // 返回访问路径
    String url = "http://localhost:8080/uploads/" + fileName;
    return Result.success(url);
}
```

#### 选项2：注册阿里云 OSS
1. 访问：https://www.aliyun.com/product/oss
2. 注册账号（新用户有免费额度）
3. 创建 Bucket
4. 获取 AccessKey
5. 配置到项目中

### 4. 文件大小限制

Spring Boot 默认文件上传大小限制为 1MB，如需上传更大文件：

```properties
# application.properties
# 单个文件大小限制
spring.servlet.multipart.max-file-size=10MB
# 总请求大小限制
spring.servlet.multipart.max-request-size=10MB
```

## 🔍 常见问题

### 1. 上传失败：400 Bad Request
- 检查参数名是否为 `image`
- 检查是否选择了 File 类型（不是 Text）
- 检查 Content-Type 是否为 multipart/form-data

### 2. 上传失败：401 Unauthorized
- 检查是否添加了 token
- 检查 token 是否有效

### 3. 上传失败：500 Internal Server Error
- 检查阿里云 OSS 配置是否正确
- 检查 AccessKey 是否有效
- 检查 Bucket 是否存在
- 查看后端控制台日志

### 4. 图片 URL 无法访问
- 检查 Bucket 是否设置为公共读
- 检查 URL 格式是否正确
- 检查阿里云 OSS 是否欠费
